Instruction for Windows

1. Install Python3

2. Install PostgreSQL and create a database

3. Run "pip install pytwitterscraper" "pip install psycopg2" and "pip install config"

4. Create db.ini and fill it out with your datqabase connection info:
[postgresql]
host=
database=
user=
password=

5. Run "python3 create_table.py"

6. Run "python3 twitter.py" to start scraping
